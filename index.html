Training Engine

Phase one is not huge-model pretraining. It starts with RAG, data cleaning, inference, evaluation, and LoRA/QLoRA-sized tasks.

Node Tiers

High-end GPUs handle heavier jobs. Normal GPUs handle inference and images. Weak devices handle cleaning and evaluation.

Model Merge

Nodes finish small jobs, upload verified updates, and the system merges them into a new version.
