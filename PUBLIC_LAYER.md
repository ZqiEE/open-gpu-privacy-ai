# Public Layer

This repository should contain only public-facing material.

Allowed:

- README
- public roadmap
- demo code
- mock adapters
- SDK examples
- public interfaces
- non-production tests

Keep out:

- production models
- production data
- private configs
- private deployment files
- runtime databases
- operator credentials
- core business logic
- private scoring logic
- private routing logic

Use this repository as the public shell, not the private engine.
