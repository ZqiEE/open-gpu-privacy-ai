# Ailovanta v1.11.0 — Guest Chat Core Preview

Release type: preview baseline

## Summary

Ailovanta v1.11.0 introduces a guest-first chat MVP. Users can open the app and start chatting without login, payment, wallet connection, or an access gate.

This release is not claiming a finished global distributed AI network. It is a working local preview for guest chat, persistent conversations, conversation context, runtime routing primitives, usage records, and reputation endpoints.

## Main user path

```text
1. Open /app.
2. Start chatting immediately.
3. A guest_id is created in the browser.
4. /ailovanta/v1/chat creates or reuses a conversation_id.
5. Messages are saved.
6. Recent history is used for follow-up turns.
7. User can load, delete, clear, or copy conversations.
```

## Main endpoints
